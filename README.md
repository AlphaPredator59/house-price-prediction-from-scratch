# House Price Prediction From Scratch

Implemented a Multiple Linear Regression model from scratch using NumPy to predict housing prices.

## Features
- Vectorized Gradient Descent
- Feature Normalization
- Train/Test Split
- R² and RMSE Evaluation
- Cost Function Visualization

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Results
Achieved an R² score of approximately 0.71 on the housing dataset.
